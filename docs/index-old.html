The Integer Book

Download the latest version of the integer book here, or browse individual sheets here.

The Integer Book is a collection of sheet music visualisations that uses numbers to refer to the relative position of notes and chords within the key of the song instead of letters referring to absolute heights.

Conventional sheet music notation The letters C, G, and A refer the absolute heights.

Integer Book notation The numbers refer the to the relative position within the key of the song (in this case C).
